# MongoDB AI Productivity Tracker
This is a MongoDB-based project created to store and analyze student productivity data such as study sessions, sleep records, and test scores.
The goal of this project is to understand how NoSQL databases work and how structured JSON data can be used for basic analytics.

## About This Project
In this project, I used MongoDB to design a flexible data model for tracking:
- Study sessions
- Sleep data
- Test scores
Instead of using tables like MySQL, this project uses collections and documents in JSON format.
This helped me understand:
- Embedded documents
- Arrays in MongoDB
- Basic queries using `find()`
- Filtering nested fields
- Simple data analysis queries
